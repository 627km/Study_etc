# 1. 제목(Header)

* 글머리 : 1 ~ 6 까지만 지원가능하다.

```
# Header<h1>
## Header<h2>
### Header<h3>
#### Header<h4>
##### Header<h5>
###### Header<h6>
```

# Header<h1>

## Header<h2>
### Header<h3>
#### Header<h4>
##### Header<h5>
###### Header<h6>

####### Header<h7>  (지원하지 않는다.)

# 2. 수평선 (Horizontal Rule)

* 각 기호를 3개 이상 입력

```
--- 
(hyphens)
*** 
(asterisks)
___ 
(underscores)
```

---

(hyphens)

***

(asterisks)

___

(underscores)
