# 1. 제목(Header)

* 글머리 : 1 ~ 6 까지만 지원가능하다.

```
# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6
```

# This is a H1
## This is a H2
### This is a H3
#### This is a H4
##### This is a H5
###### This is a H6

####### This is a H7  (지원하지 않는다.)

# 2. 수평선 (Horizontal Rule)

* 각 기호를 3개 이상 입력

```
--- 
(hyphens)
*** 
(asterisks)
___ 
(underscores)
```

---

(hyphens)

***

(asterisks)

___

(underscores)
